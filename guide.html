玩法導覽、成長方向與重點系統（以伺服器實際設定為準）

核心玩法：Mine and Slash（挖礦與砍殺）

  • 等級與成長：主世界刷怪、探索地城、挑戰頭目都會推動你的角色強度上升。
  • 裝備詞綴與稀有度：掉落裝備的詞綴與屬性會明顯影響輸出、生存與移動效率，是本服最重要的強化來源。
  • 流派選擇：近戰／遠程／法術皆可成形。建議先把一條路線練穩（生存＋輸出）再擴展其他玩法。
  • 刷裝循環：打怪 → 掉裝 → 換裝 → 挑更高難度內容 → 再掉更好裝備，是本服主要遊玩節奏。

新手前期建議（最穩路線）

  1. 先建立安全基地：床、箱子、熔爐、基本食物線。
  2. 把「能活下來」放第一順位：護甲、回血手段、抗性類道具優先於追求高輸出。
  3. 先找低風險資源點：村莊、礦區、地表小型結構；避免一開始就硬闖大型地城。
  4. 穩定刷第一套可用裝備：挑你能穩定打的怪，堆出一套「能打能扛」的裝備再往上推。
  5. 準備進異世界前先補齊：回程手段、備用裝、藥水／食物、重生點。

戰鬥系統（史詩戰鬥系列 / SlashBlade）

  • 史詩戰鬥系列（Epic Fight）：本服戰鬥偏動作向，重視走位、時機與出手節奏；不建議站樁硬拼。
  • SlashBlade 系列（拔刀劍）：高機動與高爆發的武器體系，適合喜歡連段與招式操作的玩家；配裝與操作熟練度越高收益越大。
  • 打王前檢查清單：補品、逃生路線、備用遠程手段、耐久與備用裝備、回程方式。

探索與異世界（地城／維度）

本服探索內容密度高（大型地城、異世界、頭目與強化怪物並存）。 進入新區域前，務必準備：食物、重生點、回程手段、備用裝備、補品。

本服主要探索內容

  • 大型地城與結構：以 When Dungeons Arise、Dungeon Realm、古代建築系列為核心，適合刷裝與找稀有資源。
  • 多維度冒險：包含天境、暮光森林、蔚藍浩空、幽墜黑暗、終界改造、地獄擴展與海底世界等，提供長期探索目標。
  • 強敵與頭目：包含 Cataclysm、Mowzie、Infernal/Legendary、多人頭目與多種頭目模組內容，建議循序漸進挑戰。

各維度進入方式（以整合包/伺服器設定為準）

  • The Aether（天境）
    • 多數整合包為「天境傳送門」進入（常見門框材料為螢石類）。
    • 若你不確定配方：請用 JEI 搜尋 aether portal 或 portal 關鍵字。
  • Twilight Forest（暮光森林）
    • 通常為「暮光傳送門」進入（常見為 2×2 水池＋自然素材裝飾＋啟動物品）。
    • 暮光多為「進度推進型」內容，建議依提示或地圖線索循序攻略。
  • Blue Skies（蔚藍浩空）
    • 通常透過專屬傳送門/核心方塊或門框進入（常見有兩個維度）。
    • 找不到入口請用 JEI 搜尋 portal / keystone / gate。
  • Deeper and Darker（幽墜黑暗）
    • 通常與深暗之域探索相關，入口多在深層探索/特定結構附近。
    • 建議準備夜視、抗性、逃生方案與高生存裝備再進行。
  • Even Better Nether（更好的地獄）／Nullscape（空無深景）
    • 入口仍使用原版方式進入地獄/終界，但地形與威脅會更複雜。
    • 建議把裝備詞綴與補品準備到位後再推進。
  • Oceanic Realms（海底世界）
    • 入口方式依整合包設定差異可能較大：若不確定，優先用 JEI 搜尋 oceanic / portal / key。
    • 建議必備：水下呼吸、夜視、移動加成或水下戰鬥方案。
  • Dungeon Realm（地城領域）
    • 你目前設定為：透過掉落/戰利品取得地圖，搭配地圖裝置進入。
    • 地城怪物密集，建議先整理背包空間、帶足補給與回程手段。

最實用：找不到入口怎麼辦

  • JEI 關鍵字搜尋：portal / gate / key / keystone / frame / catalyst
  • 看模組指南書：部分模組會提供指南書或進度提示（例如暮光森林）。

科技與自動化（Mekanism / AE2 / Create）

  • Mekanism（通用機械）：資源處理與發電體系完整，前中期能大幅提升挖礦與材料產出效率。
  • Applied Energistics 2（AE2）：中後期倉儲與物流核心，適合解決「箱子爆炸」與材料管理問題。
  • Create（機械動力）：偏機械結構與物流自動化，適合喜歡造機械與產線的人。
  • Storage Drawers / Iron Chests / Sophisticated Backpacks：在 AE2 前的過渡倉儲與外出整理能力。

魔法與成長（Ars Nouveau / Forbidden & Arcanus / 匠魂 / 神秘農業）

  • Ars Nouveau（新藝術）：功能與戰鬥法術兼具，建議先用「移動、採集、保護」解決痛點，再追求輸出。
  • Forbidden and Arcanus（禁忌與奧術）：提供特殊資源、儀式與裝備方向，適合中期拓展。
  • Tinkers’ Construct 3（匠魂3）：模組化鍛造與工具武器養成，適合追求手感與客製化。
  • Mystical Agriculture（神秘農業）：資源種植與穩定產出，適合想把材料供應變成「長期自給」的玩家。
  • Enchanting Infuser（附魔注入器）：提供另一種附魔獲取/強化路線，減少純 RNG 壓力。

推薦你下一步要做什麼（依你遇到的問題選）

  • 常死、打不贏：先補防禦詞綴、回血手段與補品，再推更高難度地城。
  • 材料不夠用：優先建立 Mekanism 基礎處理線（雙倍/多倍產出）與穩定電力。
  • 箱子滿到爆：先用 Storage Drawers / Iron Chests 過渡，再往 AE2 倉儲體系走。
  • 想衝頭目與高難度：先把「補給、回程、撤退方案、備用裝」準備齊，再挑戰 Cataclysm / Mowzie 等高風險內容。
  • 想玩長期自給：往神秘農業與自動化產線發展，讓補品與材料供應穩定。
